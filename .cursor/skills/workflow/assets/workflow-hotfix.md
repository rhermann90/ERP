# Hotfix Workflow

Urgent flow for production hotfixes.

## Steps

1. **Implement** — Implement the fix.
2. **/code-review** — Paranoid review. (Cursor may show as /review.)
3. **/ship** — Land the branch.

## Usage

Run `/workflow` and select "hotfix", or create `.cursor/skills/hotfix-workflow/SKILL.md` with this content and run `/hotfix-workflow`.
